# Portfolio de Data Analyst

Bienvenue sur mon portfolio de Data Analyst ! Ce projet vise à présenter mes compétences, expériences et projets dans le domaine de l'analyse de données.

## À propos de moi

Passionnée par le Data, j'aide à résoudre vos problématiques et  prendre les bonnes décisions, basées sur des données. Je traduis les données en informations précieuses et compréhensibles. Mon objectif est d’extraire une information statistique qui permet de cerner plus précisément le profil de la donnée.  Les résultats obtenus permettent ensuite d’optimiser la stratégie de la société,  d'améliorer les bénéfices d’une entreprise, de prendre les bonnes décisions, d’anticiper les potentiels risques et de réduire les coûts

## Compétences

- **Langages de programmation :** Python, SQL
- **Outils d'analyse de données :** Pandas, NumPy, Matplotlib, Seaborn, Bokeh
- **Base de données :** MySQL, PySpark
- **Visualisation des données :** Tableau, Power BI
- **Langages Web :** HTML, CSS (pour la création de rapports et de tableaux de bord)

## Projets

### Projet 1: Estimation de ventes e-commerce

- **Description :** Analyse des ventes l'une des plus grandes plateformes de commerce électronique au Pakistan. Mon objectif est de sortir l'information importante pour le conseil d'administration de l'entreprise et les aider à prendre les décisions pertinentes.
- **Outils utilisés :** Python (Pandas, Matplotlib),Jupyter Notebook, Streamlit
- **Lien vers le projet :** https://annettedpro-ecom-app-lb5wlj.streamlit.app/

### Projet 2: Création Dashboard des ventes Power BI

- **Description :** L'objectif d'un tableau de bord est de visualiser d'un seul coup d'œil les indicateurs clés de performance et les résultats les plus importants. Les tableaux de bord sont un moyen interactif de voir comment l'entreprise ou le projet fonctionne. 
- **Outils utilisés :** Power BI, DAX
- **Lien vers le projet :** [Exploration du marché immobilier](lien_vers_le_projet_2)
